# custom-template

## 从0自定义配置react项目模版

### 1. 安装依赖
```bash
npm install
```

### 2. 启动项目
```bash
npm run dev
```

### 3. 打包项目
```bash
npm run build
```

## 框架
使用 React

## 组件
使用 antd

## 路由
使用 React Router

## 路由懒加载
使用 loadable-component

## 状态管理
使用 Jotai

## 样式
使用 less
